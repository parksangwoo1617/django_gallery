# Django Gallery

**🌐 라이브 사이트**: [http://django-gallery.kro.kr/](http://django-gallery.kro.kr/)

## 🎯 주요 기능

- **작가 관리**: 작가 신청 및 승인 시스템
- **작품 관리**: AWS S3를 통한 이미지 업로드 및 관리
- **전시회 관리**: 작가별 전시회 개최 및 운영
- **관리자 대시보드**: 통계 및 신청 관리
